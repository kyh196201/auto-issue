# auto-issue

## Install & Development

```bash
npm i

# npm 배포
npm publish --access public

# auto-issue help
npm run dev -- help

# auto-issue setup
npm run dev -- setup

# auto-issue clear
npm run dev -- clear

# auto-issue start
npm run dev -- start
```

## Use

```bash
npm install -g @seungwoo-kim/auto-issue

which auto-issue

auto-issue help
```
